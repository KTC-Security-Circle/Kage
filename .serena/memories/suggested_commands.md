# Suggested Commands
- `uv run poe sync` — install/refresh project dependencies.
- `uv run poe fix` — run Ruff format + lint (auto-fix) across the repo.
- `uv run poe check` — run lint/type checks without fixing.
- `uv run poe test` — execute pytest suite.
- `uv run poe run` — launch Windows desktop app.
- `uv run poe web` — launch web front-end.
- `uv run poe cli` — run CLI harness for logic testing.
- `uv run <path/to/script.py>` — execute individual entry scripts (e.g., agents).
